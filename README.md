# Castware
A simple Javascript broadcast browser library for inner-tab communication.

## Installation 
```sh
npm install castware --save
yarn add castware
```

## Usage

### JavaScript

```javascript
var cast = require('castware');
```

### TypeScript
```typescript
import { cast } from 'castware';
```

### AMD
```javascript
define(function(require,exports,module){
  var cast = require('castware');
});
```

## Development 
```sh
npm run test
```
